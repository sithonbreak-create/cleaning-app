# SithonBreak Cleaning System

## Deploy στο Render
1. Ανέβασε όλα τα αρχεία στο GitHub repository.
2. Render > New Static Site.
3. Build Command: `npm run build`
4. Publish Directory: `dist`

## Κωδικοί
- Admin: admin / admin2026
- Πελάτης: client / client2026
- Δικά σου: sithon / sithon2026
- Καθαρίστριες: vrachos / vrachos2026, inlofts / inlofts2026, theros / theros2026, okkio / okkio2026

## Excel
Ανεβάζεις .xlsx ή .xls με στήλες:
- Room ή Κατάλυμα
- Check-in
- Check-out
- Όνομα (προαιρετικά)

Για τις 4 μονάδες, το app διαβάζει και τα sheets:
- Vrachos Lofts, Άφυτος
- InLofts, Άφυτος
- Okkio Sea View, Γαλήνη
- THEROS APARTMENTS
